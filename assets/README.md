# Assets Directory Structure

画像ファイルの配置場所

## ディレクトリ構成

```
assets/
├── works/
│   ├── highlights/  # Recent Highlights用の画像
│   │                # (プロジェクトのサムネイル画像など)
│   ├── sketches/    # Daily Sketches用の画像
│   │                # (日々のスケッチのサムネイル)
│   └── led/         # LED Works用の画像
│                    # (PicoRuby LED作品の画像)
└── activities/      # Activities用の画像
                     # (将来的なイベント・活動記録の画像)
```

## 画像配置ガイドライン

### Recent Highlights (`works/highlights/`)
- プロジェクトのサムネイル画像
- 推奨サイズ: 1200x800px以上
- フォーマット: JPG, PNG

### Daily Sketches (`works/sketches/`)
- 日々のスケッチのサムネイル
- 推奨サイズ: 800x800px以上（正方形）
- フォーマット: JPG, PNG

### LED Works (`works/led/`)
- PicoRuby LED作品の画像・動画
- 推奨サイズ: 800x800px以上（正方形）
- フォーマット: JPG, PNG, MP4

### Activities (`activities/`)
- イベント・ワークショップの写真
- フォーマット: JPG, PNG
