# 展示作品画像フォルダ

このフォルダには展示作品の写真を配置します。

## 画像仕様

### サイズと縦横比
- **縦横比**: 1:1（正方形）推奨
- **推奨サイズ**: 800×800px ～ 1200×1200px
- **最小サイズ**: 600×600px
- **最大サイズ**: 1500×1500px程度

### フォーマット
- **JPG**: 写真の場合（ファイルサイズが小さい）
- **PNG**: 透過が必要な場合
- **WebP**: さらに最適化したい場合

### ファイル名の例
```
tangible-code.jpg
spring-has-come.jpg
fractal-flower.jpg
work-2025-01.jpg
```

## 正方形以外の画像について

縦長・横長の画像も使用できますが、表示時に自動的にトリミングされます：
- 中央部分が優先的に表示されます
- 重要な要素が画像の中央にあることを確認してください
- 余白を含めて正方形に加工することを推奨します

## 画像の最適化

### ファイルサイズの目安
- 1枚あたり100KB ～ 300KB程度を推奨
- 最大でも500KB以下に抑えると読み込みが速くなります

### 最適化ツール
- [TinyPNG](https://tinypng.com/) - JPG/PNG圧縮
- [Squoosh](https://squoosh.app/) - 各種フォーマット対応
- ImageMagick (CLI): `magick convert input.jpg -quality 85 -resize 1000x1000 output.jpg`
